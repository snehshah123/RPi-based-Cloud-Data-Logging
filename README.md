# RPi-based-Cloud-Data-Logging
RPi Cloud Data Logger 📊🌩️  A Raspberry Pi-based data logging system using RS-485/USB to collect sensor data and upload it to the cloud (Google Drive, AWS, or a custom server). Supports multiple Modbus scanners, auto-detection, real-time CSV logging, and Google Looker Studio integration for visualization. Runs on boot for seamless operation. 🚀
